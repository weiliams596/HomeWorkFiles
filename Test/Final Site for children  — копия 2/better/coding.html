Бағдарламалау тілдері

CSS

CSS (Cascading Style Sheets) – сайттың сыртқы көрінісін безендіруге арналған тіл. Ол түстерді, шрифттерді және орналасу дизайнын басқарады.

HTML

HTML (HyperText Markup Language) – веб-беттердің құрылымын жасауға арналған белгілеу тілі. Ол сайттың мәтінін, суреттерін және сілтемелерін анықтайды.

JavaScript

JavaScript – веб-беттерді интерактивті етуге арналған бағдарламалау тілі. Онымен анимацияларды, батырмалар әрекетін және ойындарды жасауға болады.

Бағдарламалау тілдеріне мысал

1-сурет: HTML Бұл үйдің қаңқасы ғана – тек ағаш бөренелерден жасалған негіз.
Қабырғалар, есік, терезелер жоқ, тек құрылымдық қаңқасы бар.
Бұл HTML-ға ұқсайды, өйткені HTML веб-беттің негізгі құрылымын құрайды, бірақ әлі сәнделмеген және функциялары жоқ.

2-сурет: CSS Бұл үйге қабырғалар, есік және терезелер қосылды, бірақ әлі толық аяқталмаған.
Бұл CSS-ке ұқсайды, өйткені CSS дизайн мен стиль береді, үйге түс,
пішін және орналасу сияқты көрнекі элементтерді қосады.

3-сурет: JavaScript Бұл үйге жарық қосылды, ал мұржадан түтін шығып жатыр.
Яғни, үй “тірі” болып көрінеді, функционалдық қосылған. Бұл JavaScript-ке ұқсайды,
өйткені JavaScript бетке интерактивтілік қосады, мысалы, батырмалар басылғанда өзгерістер енгізу немесе анимациялар жасау.

4-сурет: React Бұл толықтай аяқталған, заманауи, сәнді және әдемі үй. Бәрі үйлесімді, тиімді, әрі көркем.
Бұл React-ке ұқсайды, өйткені React веб-қосымшаларды тиімді құруға мүмкіндік береді,
қолданушыға ыңғайлы интерфейстер жасайды және веб-бетті толық функционалды етеді.
